การวางแผนตามเป้าหมายที่สร้างไว้ตามหลักPDCA
1.Plan
เป้าที่สร้าง
-ไม่ติดFซักวิชา
-เรียนหาความรู้แล้วทำproject อย่างน้อย1project(project Game by java) ตัวชี้วัด
-ผลคะแนนของแต่ละเทอม
-projectที่เสร็จเรียบร้อย และ คะแนนสอบ เรื่อง java และ OOP
2.Do
-ลงมือทำพยายามตั้งใจฟังในคาบบรรยาย โดยเฉพาะหัวข้อที่สำคัญ พยายามคิดตามที่ผู้สอนพูด และไม่จดในคาบบรรยาย แต่อาจจดสรุปทีหลังว่าที่ฟังมีอะไรสำคํญและมีคำถามอะไรที่ต้องหาคำตอบเพื่มเติมจาก แหล่งข้อมูลอื่นหรือchatGPT
-ตังใจเรียนเรียนรู้เรือง java OOP ในคาบ Computer programming2 และวางแผนทำprojectเกม เนื่องจากต้องส่งเป็นfinal projectของวิชานี้ด้วย
3.Check
-มีเรื่องที่ยังไม่เข้าใจในบทเรียนหรือไม่ มีบางวิชาที่ยังไม่เข้าใจหรือเปล่า
-ระหว่างทำproject Game Programที่เขียนไว้ทำงานได้ตามที่ต้องการหรือไม่ มีข้อผิดพลาดหรืออะไรที่ทำให้programทำงานได้ไม่ดีเท่าที่ควรหรือเปล่า
4.Act -ทบทวนบทเรียนหาข้อมูลเพิ่่มเติมจนมั่นใจว่าเขาในเรื่องเขิงลึกพอสมควร
-debug และทดสอบgame อาจลองส่งให้เพื่อนเล่นเพื่อรับข้อคิดเห็นและปรับปรุงอีกครั้ง
-เนื่องจากเป้าหมายที่วางไว้มันค่อนข้างกว้างและตอบได้ยากว่าอะไรจะเปลี่ยนไปบ้าง อาจเปลี่ยนจากตั้ังเป้าหมายใน2025-2030 เป็นเป้าหมายย่อยในแต่ละปี
และใช้หลัก PDCAในการติดตามและประเมินต่อไป
